# Database with Reflection Project

This is a project which uses Reflection to access and modify a database. It mainly uses Java and consequently, JavaFX for the interface, while MySQL has been used for the database. The application is structured into multiple layers: Models, Data Access Objects (DAOs), Business logic and the GUI. This makes the application modular, easy to understand, and as effortless as possible to maintain. Connecting the user to the app, the User Interface is simple but clear and easy to use.
